# Groww-Mobile-Website-Project
GROWW mobile website application to query about mutual funds and other investment options.

## Steps to start application
1. git clone
2. npm install
3. npm run build-dev
4. npm run start-dev
5. Your app will be running on 0.0.0.0:4000
